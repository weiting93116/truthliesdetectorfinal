首頁 系統架構 操作介面 團隊成員
MODULE 01 / AUTH & HOME

帳戶管理與首頁

提供流暢的登入與註冊介面。
首頁匯集了最新與熱門的訊息動態，讓用戶在進入 App 時能迅速瀏覽當前備受關注的議題資訊。

MODULE 02 / AI ASSISTANT

AI 助手與趨勢報告

透過 AI 對話介面與搜尋，系統能針對特定議題進行分析。
除了即時的辨識結果，還提供每周報告功能，協助用戶回顧與追蹤近期資訊環境的變化趨勢。

MODULE 03 / CONTENT VIEW

文章內頁與可信度標籤

在詳細的文章閱讀介面中，系統會直接標註該內容的可信度等級。
透過直觀的視覺標籤提示，幫助用戶在閱讀過程中心中有一把尺，初步判別訊息風險。

MODULE 04 / FLOATING BALL

智慧懸浮球功能

此功能支援在手機其他介面中隨時呼叫，無需頻繁切換程式。
透過懸浮球，用戶能快速開啟常用工具或進行簡單查核，提升操作效率與便利性。

MODULE 05 / SETTINGS

個人化設定介面

用戶可依據喜好調整帳戶資料與 App 的運作。
從關注議題到收藏，多層次的設定選項讓真偽探測站能更貼合每一位使用者的日常習慣。
